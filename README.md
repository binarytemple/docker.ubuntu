docker role
===========

This role adds the docker PPA, and installs the docker package - if 
`/usr/bin/docker` does not already exist.

Requirements
------------

Ubuntu only

Role Variables
--------------

* docker_package_name: lxc-docker - name of the Ubuntu package

Dependencies
------------

None
